## рџ“– Project Overview

This repository contains a complete **3D Stop Valve Assembly** designed using **SolidWorks**. The project demonstrates parametric part modeling, mechanical assembly techniques, and engineering design principles used in industrial piping systems.

A stop valve is used to regulate or completely stop the flow of fluids in pipelines. This assembly consists of individually modeled components assembled using SolidWorks mates to create a functional valve mechanism.
